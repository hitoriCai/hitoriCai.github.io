SJTU 不断按回车键, 退出当前列表 ## 9. *上下标 (扩展语法, 其他编译器可能不支持) #### 上标 用 "^" 包围, 只有一层上标, 没有"上标的上标" 欢迎来到^SJTU^ 欢迎^来到^SJTU #### 下标 用 "~" 包围, 同理只有一层 欢迎来到~SJTU~ 欢迎~来到~SJTU ## 10. *注释 (扩展语法, 其他编译器可能不适配) 语法: ``` 欢迎来到SJTU![^1] [^1]: 海上浇透大学 ``` eg: 欢迎来到SJTU![^1] [^1]: 海上浇透大学 ## 11. 链接 ``` [文本](链接) ① [github中的markdown](https://github.com/Sakiyary/Markdown-Typora-VSCode-Doc) ② [跳转到文内的某个目录(扩展语法)](##1. 标题) 注:②中括号的内容必须和想去的标题 完全一样 ``` [github中的markdown](https://github.com/Sakiyary/Markdown-Typora-VSCode-Doc) 按着Ctrl点题目 [教程---1. 标题](##1. 标题) 按着Ctrl点题目 ## 12. 任务列表 to do list. ``` - [ ] 任务一 - [x] 任务二 (代表勾选上) ``` - [ ] 任务一 - [x] 任务二 ## 13. 表格 ``` |栏目1|栏目2|栏目3| | :- | :-: | -: | (引号的位置表示: 左对齐 居中 右对齐)(这块左右的空格必须有) |SJTU|电院|IEEE| ``` |栏目1|栏目2|栏目3| | :- | :-: | -: | |SJTU|电院|IEEE| ## 14. 图片 直接拖进来 / 复制粘贴 / ![图片的位置] **注意: 删除那个文件夹里的图片的话, 笔记中的图片也会一并消失!** 删掉这里的链接就相当于在md中删除了图片, 但是相应文件夹中的图片是不变的! ![image-20230203155052873](markdown测试学习.assets/image-20230203155052873.png) 这里指把粘贴进来的图片, 保存在一个和md文件**同层**的新建文件夹里 如果要把md文件发给别人, 必须把这个含有图片的文件和md文件打包, 一起发过去 缺点：不灵活不好分享，本地图片的路径更改或丢失都会造成markdown文件调不出图。 ![image-20230203155834453](markdown测试学习.assets/image-20230203155834453.png) ## 15.分割线 三个 "*" 或 "-" 或 "_" 并回车 --- ## 16. emoji (英文输入为扩展语法) ``` :sweat_smile: :drooling_face: :clown_face: (敲回车或鼠标点击, 后面的":"最好写上) ``` :sweat_smile::drooling_face::clown_face:可以放到一行, 首尾都加上冒号就行哩 对于一般的编辑器, 直接复制emoji进来就行! 万能! 🤣 [全emoji网站! 可以直接复制粘贴!](https://emojipedia.org/) # 进阶教程🤓 ## 1. 目录 ``` [TOC] (此为typora特有的，例如本文档开头) ``` 若使用VS Code搭配Markdown All in One扩展, 可在VS Code的 `命令面板` (即VS Code Command Palette) 输入`Create Table of Contents` 自动生成目录, 且可以在扩展设置中细挑目录参数. ## 2. 内联HTML代码 完全可以吧md当html来写 eg: ①居中: 用`
